# Название Telegram бота: KinoManBot

**Краткое описание**: Telegram бот для поиска фильмов с синхронизацией и базой данных.  
**Публичная ссылка в Telegram**: @My_FirstTrialBot

## Описание
Бот ищет фильмы по названию, по жанру, по рейтингу, фильмы с низким бюджетом, фильмы с высоким бюджетом. 
Сохраняет историю поиска в Peewee ORM и по запросу пользователя может показать историю поиска за выбранную дату.
Так же поддерживает inline-кнопки и пагинацию результатов поиска.

## Старт
**Версия Python**:

    Python 3.13

**Создаем виртуальное окружение**:

    python -m venv venv

**Активация виртуального окружения**:  

    venv/Scripts/Activate.ps1

**Устанавливаем все зависимости в Python из файла requirements.txt**  

    pip install -r requirements.txt

**Содержание файла requirements.txt**:  

    pyTelegramBotAPI==4.29.1  
    python-dotenv==1.2.1  
    requests==2.32.5  
    peewee==3.15.4
